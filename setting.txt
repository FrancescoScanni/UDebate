
SET UP -> Win Powershell/Git Bash

1.  (SOLO SU PWSD)

2. npm install (installare l'ultima versione di Node.js)

3. npm run build 

4. cp .env.example .env (basato su .env.example standard di laravel)

5. docker compose up -d

6. docker compose exec app composer install

7. docker compose exec app php artisan key:generate

8. docker compose exec app php artisan migrate

9. Vai su docker e localhost:8080 browser
